# Data Structures & Algorithms Preparation

This folder contains practice problems for data structures and algorithms commonly asked in technical interviews.

## Topics Covered

### Arrays
- Array manipulation
- Two pointers technique
- Sliding window problems
- Prefix sum and difference arrays

### Strings
- String manipulation
- Pattern matching
- String parsing
- Palindromes and anagrams

### Linked Lists
- Singly and doubly linked lists
- Fast and slow pointer technique
- Reversal and reordering
- Cycle detection

### Trees
- Binary trees and BST
- Tree traversals (inorder, preorder, postorder)
- Balanced trees (AVL, Red-Black)
- Tree construction and serialization

### Graphs
- Graph traversal (BFS, DFS)
- Shortest path algorithms
- Topological sorting
- Union-Find

### Sorting & Searching
- Sorting algorithms
- Binary search and variations
- Search in rotated arrays
- Merge intervals

### Dynamic Programming
- 1D and 2D DP
- Memoization and tabulation
- Common patterns (knapsack, LCS, etc.)
- State machine DP

### Hash Tables
- Hash map implementations
- Collision resolution
- Design problems
- Frequency counting

### Stacks & Queues
- Stack operations
- Queue implementations
- Monotonic stack/queue
- Design problems

### Heap & Priority Queue
- Min heap and max heap
- Top K problems
- Merge K sorted arrays
- Median finding

## Practice Tips

1. Start with easy problems in each category
2. Focus on understanding patterns
3. Time your solutions
4. Review multiple approaches
5. Practice explaining your solution
