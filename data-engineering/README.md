# Data Engineering Skills

This folder contains practice problems and concepts related to data engineering.

## Topics Covered

### ETL Pipelines
Practice building Extract, Transform, Load pipelines with various data sources and destinations.

### Data Warehousing
Explore data warehouse design patterns, dimensional modeling, and OLAP concepts.

### Streaming
Work with real-time data processing using technologies like Kafka, Spark Streaming, and Flink.

### Batch Processing
Practice batch data processing with frameworks like Apache Spark, Hadoop, and cloud-native solutions.

### Data Quality
Implement data validation, cleansing, and monitoring strategies.

### Database Design
Practice database schema design, indexing strategies, and query optimization.

### Distributed Systems
Explore concepts related to distributed data processing and storage systems.

## Getting Started

Each subfolder contains specific exercises and reference materials for that topic. Start with the basics and progress to more advanced concepts.
